## 🛠️ 작성 규칙

- 배운 것이라면 주제는 무엇이든 상관 없다.
    - 책에서 얻은 배움, 강의를 보고 따라한 클론코딩, 타 개발자를 통해 얻은 인사이트 등
- 하루 동안 배운 것을 기록하고 거짓된 기록은 하지 않는다.
    - 삽질한 경험, 헤멘 경험도 솔직하게 적는다.
- 미래의 내가 알아볼 수 있도록 친절하게 적는다.
